> [!IDEA]
> [[Graph (Mathematical)]] with no cycles

### Properties
- [[Connected Graph]]
	- For any [[Cut (Mathematical)]], must have at least one [[Crossing Edge]] ^df6ba1
- There is at most one path between any pair of vertices.
### Examples
- [[Merkle Tree]]
- [[Suffix Tree]]
- [[Minimum Spanning Tree]]
- [[Trie]]
